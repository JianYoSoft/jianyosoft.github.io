简游配置文件编辑器

M2StringEditer 是一款快速修改 MirServer 配置文件的工具。

支持如下文件:

- 服务端字符串配置文件(String.ini)

- 服务端运行配置文件(!Setup.txt)

- 游戏命令配置文件(Command.ini)

- 服务端经验表配置文件(Exps.ini)

- 服务端变量配置文件(GlobalVal.ini)

系统要求:

推荐: Windows 7 x64 SP1 或 Windows Server2008 R2 及更高版本。

兼容: Windows XP SP3 或 Windows Server 2003 R2。

简游配置工具

M2ConfigTool 是一款轻量的 MirServer 版本配置工具。

在绝大多数情况下,它都能很好的完成你对 MirServer 的配置操作。

软件界面通俗易懂 方便新手和爱好者的使用,并且兼容市面大多数引擎。

系统要求:

推荐: Windows 7 x64 SP1 或 Windows Server2008 R2 及更高版本。

兼容: Windows XP SP3 或 Windows Server 2003 R2。

DBC2000注册工具

RegistDBC 是一款用于注册 64位 DB Commander 2000 PRO 的小工具。

注册 DB Commander 2000 PRO 以解除30天使用限制。

系统要求:

Windows 7 x64 SP1 或 Windows Server2008 R2 及更高版本。
